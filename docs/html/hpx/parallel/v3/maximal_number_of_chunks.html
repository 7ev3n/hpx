Function template maximal_number_of_chunks

hpx::parallel::v3::maximal_number_of_chunks

Synopsis

// In header: <hpx/parallel/executors/executor_parameter_traits.hpp>


template<typename Executor> 
  std::size_t maximal_number_of_chunks(executor_parameters_type & params, 
                                       Executor & exec, std::size_t cores);

Description

Return the largest reasonable number of chunks to create for a single algorithm invocation.

Parameters:

cores

[in] The number of cores the number of chunks should be determined for.

exec

[in] The executor object which will be used used for scheduling of the the loop iterations.

params

[in] The executor parameters object to use for determining the number of chunks for the given number of cores.
